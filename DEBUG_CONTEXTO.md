## Regras de Refinanciamento de Carteira - BMG

### Requisitos
- Contrato ativo no BMG
- Mínimo de **6 parcelas pagas**
- Contrato sem atraso

### Taxa
- Taxa máxima: **1,84% a.m.**

### Prazo
- Prazo máximo: **84 meses**
- O prazo pode ser maior que o remanescente do contrato original

### Troco
- O refinanciamento pode gerar troco positivo para o cliente
- Valor mínimo de troco: verificar no sistema

### Observações
- O refinanciamento substitui o contrato original
- A nova parcela pode ser igual, menor ou maior que a original (desde que caiba na margem)
- O BMG pode recusar refinanciamentos se identificar risco de crédito

## **FORMALIZAÇÃO DIGITAL**

Para oferecer essa nova facilidade, você parceiro **BEVI**, deverá cadastrar as operações no Agente Facta. Ao final da operação, terá um link que poderá ser enviado por SMS ou WhatsApp do cliente.

Confira o passo a passo:

1**.** Assinatura de Contrato: É onde tudo começa. Aqui, o usuário deve apenas clicar em "continuar".

![Imagem.png](/hc/article_attachments/4412582059927)

###### 

Algumas **formalizações digitais** podem apresentar o **Banco PINE** no momento em que o cliente acessa o **link de assinatura**.

Isso ocorre porque, nas operações de **Port + Refin** e **Novo**, a formalização é feita em nome do **Banco PINE**.  
Já nas **demais operações**, o processo segue normalmente como **Banco Facta**.

|  |
| --- |
| Formalização banco Pine.png |

2. **Termo e condições de Uso:** Nessa tela, o cliente toma seu tempo para analisar os termos do contrato. Quando estiver satisfeito, o usuário clica em "eu li e aceito os termos" para prosseguir.

|  |  |
| --- | --- |
| Imagem.png | |

3. **Dados e Condições da proposta:** Aqui, o cliente confirma valores e dados cadastrais, além das condições gerais da proposta. Se estiver tudo de acordo, o usuário clica em "Li e estou de acordo com a proposta."

![Imagem.png](/hc/article_attachments/4412587193623)

4. **Documentos:** Agora, é necessário fazer upload de fotos de um documento de identificação do cliente (CNH ou RG), e demais documentações de acordo com cada convênio (vide roteiro operacional). 

Obs: Se for beneficiário com rep. legal, deve ser encaminhado o documento de identificação do representante legal.

***Importante:** Neste momento do envio do documento de identificação, não serão aceitos documentos digitais. Apenas o documento físico será aceito. Documentos digitais são permitidos apenas posteriormente, no andamento da proposta, para resolução de pendências.*

**Importante!**  
Para clientes que possuem contratos pagos nos últimos 12 meses com o Banco Facta, e tiverem os documentos aprovados conforme os critérios de validação do Banco, serão ISENTOS de envio dos documentos no momento da formalização.

![Imagem.png](/hc/article_attachments/4412582538007)

Atenção! 

Caso o sistema não consiga validar o documento do cliente, será apresentada uma mensagem de erro solicitando um novo envio. Será possível selecionar novamente qual documento será enviado como Doc. de Identificação:

|  |  |
| --- | --- |
|  |  |

5. **Selfie:** (Exclusivo modalidade não presencial) 

São 2 momentos: 

* Primeiro acontece o reconhecimento facial, onde o cliente deve estar com a expressão neutra.
* Imediatamente após a captura dessa, o cliente deve sorrir.

Obs: Se for beneficiário com rep. legal, deve ser encaminhado a selfie do representante legal.

|  |  |
| --- | --- |
| Imagem.png | |
| Imagem.png | |

06. **Áudio**: Por último, o cliente grava um áudio dentro da própria plataforma. As instruções do que deve ser dito aparecem na tela". Para clientes que recebam por cartão Magnético, ou de cidades fronteiriças.

![Imagem.png](/hc/article_attachments/4412583170455)

07. **Confirmação**: Para concluir, o usuário clica em "confirmar a assinatura do contrato" e, em seguida, em ok".

![Imagem.png](/hc/article_attachments/4412593727127)

Pronto! Ao final do processo, surge uma tela que confirma o fechamento de contrato e apresenta canais de comunicação para eventuais dúvidas.

![Imagem.png](/hc/article_attachments/4412593745559)

Caso seja necessário o reenvio de link ao cliente, poderá ser utilizado outro aparelho não cadastrado na proposta em caráter de tentativa.

Se o cliente não formalizar a operação em até 5 dias úteis da digitação da proposta, a operação será reprovada por decurso de prazo, sendo necessário redigitar a operação.

A localização onde o cliente realiza a formalização digital, não interfere no andamento da proposta, desde que ele esteja dentro do território Brasileiro.   
Caso o cliente esteja fora do do território Brasileiro, c[lique aqui para verificar as regras.](/hc/pt-br/articles/13428210861975-Resumo-Restri%C3%A7%C3%A3o-de-Pa%C3%ADses-#)
